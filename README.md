# Meteora

Meteora is a weather app built using React Native. It uses the Visual Crossing API to fetch and display weather data.

**Note**: The project is still under final preparations and will be updated or released soon.

## Features
- Displays current weather information based on location.
- Provides weather forecasts for the next few days.

## Installation
1. Clone this repository.
2. Install dependencies: `npm install` or `yarn install`.
3. Run the app: `expo start`.

Stay tuned for further updates and the official release.
