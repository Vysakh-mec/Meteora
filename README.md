# Meteora
Meteora is a weather app built using React Native and Expo. It allows users to check current weather conditions based on their location, providing accurate data powered by the Visual Crossing API. With a sleek and intuitive interface designed in Figma, Meteora ensures an engaging user experience.

## Features
- Current Weather: Displays real-time weather conditions for the user’s current location.
- Location Detection: Utilizes Expo Location to fetch the user’s current geographical location.
- Clean UI: Designed using Figma Community templates for a modern and user-friendly interface.
- Custom Icons: Integrated weather visuals with React Native SVG Transformer for scalable, high-quality icons.
- Navigation: Smooth navigation using React Navigation.

## Tech Stack
- Frontend: React Native, Expo
- API: Visual Crossing API for weather data
- Navigation: React Navigation
- Design: Figma Community templates
- Additional Libraries:
  - Expo Location
  - React Native SVG Transformer  

## Installation
1. Clone this repository.
    ```bash
    git clone https://github.com/Vysakh-mec/Meteora.git
    cd Meteora
    ```
3. Install dependencies: `npm install` or `yarn install`.
4. Run the App: Launch the app on an emulator or physical device: `npx expo start`.
